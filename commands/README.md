# KN Discord Bot

Bot Discord dla koła naukowego.

## Wymagania

- Python 3.10+

## Instalacja

1. Sklonuj repozytorium

git clone https://github.com/...

2. Wejdź do folderu

cd kn-algo-bot

3. Utwórz środowisko

python -m venv venv

4. Aktywuj środowisko

Linux / Mac

source venv/bin/activate

Windows

venv\Scripts\activate

5. Zainstaluj zależności

pip install -r requirements.txt

6. Skopiuj konfigurację

cp .env.example .env

7. Wklej token bota do `.env`

8. Uruchom bota

python bot.py
